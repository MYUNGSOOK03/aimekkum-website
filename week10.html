10주차: 사진이 말하는 감동 영상 만들기 (최종회)

🚀 빠른 실행

  • 준비물: PC, 사진 1~3장, 구글 계정
  • 바로가기: D-ID · ElevenLabs · Runway
  • 오늘 만들 것: 1분 내외 내레이션+영상 클립 1개(MP4)

🧰 준비물

  • 사진 1~3장(정면, 고화질 권장)
  • 스クリپ트 초안(200~300자)

🔗 바로가기

  • D-ID · ElevenLabs · Runway

🎬 옛날 사진 한 장에 목소리를 입혀보세요!
돌아가신 부모님, 고인이 된 친구, 어린 시절 나의 모습이 말을 합니다. (무료!)

🎯 이번 주 목표

  • ✅ 사진 속 인물이 말하는 영상 만들기
  • ✅ AI 음성으로 감동적인 메시지 전하기
  • ✅ 가족 영상, 추모 영상, 손주 선물 만들기
  • ✅ 10주간 배운 모든 것을 종합하기

💡 1. 어떤 영상을 만들 수 있나요?

🎬 감동적인 영상 아이디어

  • 👴 부모님 생신 영상: 돌아가신 아버지 사진이 "생신 축하해요"라고 말함
  • 💐 추모 영상: 고인이 된 가족이나 친구의 사진으로 추억 이야기
  • 💑 결혼기념일 영상: 젊은 시절 사진이 배우자에게 사랑 고백
  • 👶 손주에게 주는 영상: 할머니/할아버지 사진이 손주에게 메시지
  • 🎓 동창회 영상: 졸업사진이 친구들에게 인사
  • 🏡 가족 역사: 증조부모님 사진으로 가문의 역사 설명
  • 📜 유언 영상: 내 모습이 자녀들에게 남기는 메시지

💝 특히 돌아가신 분의 사진으로 만들면, 마치 다시 만난 것 같은 감동을 선사합니다.

AI가 만드는 영상의 종류

  • 텍스트→영상: 글 설명만으로 영상 생성
  • 이미지→영상: 정지 사진을 움직이는 영상으로
  • 영상 편집: 기존 영상을 AI로 보정, 스타일 변환
  • 립싱크: 사진 속 인물이 말하는 것처럼

🎬 무료 AI 영상 도구 총정리

🔥 초보자 추천 (무료)
  • D-ID: d-id.com - 사진 립싱크 (월 5분 무료)
  • ElevenLabs: elevenlabs.io - AI 음성 생성 (월 10분 무료)
  • Runway: runwayml.com - 영상 생성 (무료 125 credits)
💰 유료 도구
  • HeyGen: $24/월 - 아바타 영상
  • Synthesia: $22/월 - 전문가용
  • Pika: $10/월 - 텍스트→영상

🗣️ 2. D-ID: 사진이 말을 한다!

💎 D-ID란?

사진 속 인물이 실제로 말하는 것처럼 입 모양을 AI로 합성해주는 도구입니다. 발표 영상, 자기소개 영상 등에 활용 가능!

Step 1: 회원가입 (무료)

  1. d-id.com 접속 →
  2. "Try for Free" 또는 "Sign Up" 클릭
  3. Google 계정으로 간편 가입
  4. ✅ 무료: 월 5분 크레딧 (약 5개 영상)

Step 2: 영상 만들기

  1. "Create Video" 버튼 클릭
  2. 사진 선택:
    • 옵션 1: 본인 사진 업로드 (정면, 고화질)
    • 옵션 2: D-ID 제공 아바타 사용 (추천!)
  3. 대사 입력:
    • 텍스트 입력: 말할 내용 직접 입력 (한글 가능)
    • 또는 음성 파일 업로드
  4. 음성 선택:
    • 한국어 음성 여러 개 제공
    • 남성/여성, 나이대별 선택 가능
    • 미리듣기로 확인
  5. "Generate Video" 클릭 → 1~2분 대기
  6. 완성! 다운로드 또는 공유

💡 D-ID 활용 아이디어

  • 🎓 자기소개 영상: "안녕하세요, 저는 OO중학교 2학년 김철수입니다..."
  • 📚 책 소개 영상: "제가 쓴 책은 AI와 우정에 관한 이야기입니다..."
  • 🎤 발표 보조: 발표 내용을 AI 아바타가 설명
  • 🎁 특별한 메시지: 부모님/친구에게 감동적인 영상 선물
  • 📰 뉴스 앵커: 학급 소식을 전하는 뉴스 형식

🔥 실습 과제

지금 바로 만들어보세요!

  1. D-ID 가입 & 로그인 →
  2. "Create Video" 클릭
  3. 제공되는 아바타 중 마음에 드는 것 선택
  4. 대사 입력: "안녕하세요! 저는 AI 영상 크리에이터입니다. 오늘은 여러분께 AI의 놀라운 세계를 소개하겠습니다."
  5. 한국어 음성 선택 → "Generate"
  6. 완성된 영상 다운로드!

🗂 예시 프롬프트

아래를 복사해 스크립트/나레이션을 빠르게 만드세요.

  • "[대상: 할머니/가족/친구]에게 전하는 1분 내외 감사 메시지를 따뜻한 톤으로 220~260자로 써줘. '고마웠던 순간 2가지'를 포함해줘."
  • "이 스크립트를 더 간결하게 다듬고, 문장 사이 호흡이 자연스럽게 끊기도록 문장부호를 추가해줘."
  • "이 텍스트를 나레이션에 적합하도록 숫자는 한글로 바꾸고, 강조할 단어는 큰따옴표로 표시해줘."

🎙️ 3. ElevenLabs: AI 음성 생성

💎 ElevenLabs란?

텍스트를 입력하면 실제 사람처럼 자연스러운 음성으로 변환해주는 도구입니다. 오디오북, 내레이션, 팟캐스트 제작에 최적!

🚀 ElevenLabs 사용법

  1. elevenlabs.io 접속 →
  2. "Sign Up" → Google 계정 연동
  3. ✅ 무료: 월 10,000자 (약 10분 음성)
  4. "Speech Synthesis" 선택
  5. 음성 선택: 다양한 언어와 톤 제공 (한국어 포함)
  6. 텍스트 입력: 음성으로 변환할 내용 입력
  7. "Generate" 클릭 → 10초 대기
  8. 미리듣기 → 마음에 들면 다운로드 (MP3)

💡 활용 아이디어

  • 📖 오디오북: 자기가 쓴 책을 음성으로
  • 🎬 영상 내레이션: Runway/D-ID 영상에 음성 추가
  • 🎧 팟캐스트: AI와 대화하는 라디오 쇼
  • 📚 학습 자료: 교과서 내용을 음성으로 들으며 공부

📸 4. Runway: 이미지를 영상으로!

💎 Runway란?

정지 이미지를 움직이는 영상으로 변환하거나, 텍스트 설명만으로 영상을 생성하는 강력한 AI 도구입니다!

🚀 Runway 사용법 (Image to Video)

  1. runwayml.com 접속 →
  2. "Sign Up" → Google 계정 로그인
  3. ✅ 무료: 125 credits (약 15개 영상)
  4. "Create Project" 클릭
  5. 좌측 메뉴에서 "Gen-3 Alpha" 선택
  6. "Image to Video" 탭 클릭
  7. 이미지 업로드:
    • Bing Image Creator에서 만든 이미지 사용 (week 6)
    • 또는 본인 사진 업로드
  8. 프롬프트 입력 (선택사항):
    • 어떻게 움직일지 설명 (영어)
    • 예: "Camera slowly zooms in"
  9. "Generate" 클릭 → 2~3분 대기
  10. 완성된 5초 영상 다운로드!

📝 Runway 프롬프트 예시

예시 1: 카메라 움직임

Camera slowly zooms in on the person's face, gentle wind blowing through hair

(카메라가 얼굴로 천천히 줌인, 바람에 머리카락이 날림)

예시 2: 배경 변화

Sunset in the background, warm orange glow, clouds moving slowly

(배경에 석양, 따뜻한 오렌지빛, 구름이 천천히 움직임)

예시 3: 대상 움직임

Person smiles and nods gently, eyes blinking naturally

(사람이 미소 짓고 부드럽게 고개를 끄덕임, 자연스럽게 눈을 깜빡임)

💡 이미지 준비 팁

  • ✅ 고화질: 최소 1080p 이상
  • ✅ 명확한 주제: 얼굴이나 대상이 선명하게
  • ✅ 좋은 조명: 너무 어둡거나 밝지 않게
  • ✅ 깨끗한 배경: 복잡하지 않은 배경

🔥 실습 과제

프로젝트: 나만의 영상 만들기

  1. 이미지 준비:
    • 옵션 1: Bing Image Creator로 이미지 생성 (예: "귀여운 고양이가 창밖을 보는 모습")
    • 옵션 2: 스마트폰으로 멋진 풍경 사진 찍기
  2. Runway 접속 & 로그인 →
  3. Gen-3 Alpha → Image to Video
  4. 이미지 업로드
  5. 프롬프트 입력: "Camera slowly zooms in, soft cinematic style"
  6. "Generate" → 영상 완성 대기
  7. 다운로드해서 친구들에게 자랑!

🎓 졸업 메시지

Step 1: D-ID 접속 및 회원가입

  • 웹사이트: d-id.com
  • 무료 플랜: 월 5분 크레딧

Step 2: 사진 업로드

  • 정면을 바라보는 사진이 가장 좋음
  • 얼굴이 화면의 60% 이상 차지

Step 3: 스크립트 작성

스크립트 예시: 할머니께 드리는 편지

할머니, 보고 싶어요. 항상 저를 응원해주시고 따뜻하게 안아주셨던 기억이 나요. 할머니가 가르쳐주신 용기와 사랑을 잊지 않을게요. 늘 건강하시고, 제가 더 멋진 사람이 되어 찾아뵐게요. 사랑해요, 할머니.

Step 4: 음성 선택

  • 한국어 음성 다양하게 제공
  • 남성/여성, 연령대별 선택 가능
  • 톤과 속도 조절 가능

Step 5: 생성 및 다운로드

  • Generate 버튼 클릭
  • 약 1-2분 대기
  • 미리보기 후 다운로드

🎙️ 4. AI 음성 생성: ElevenLabs

자연스러운 내레이션 만들기

ElevenLabs 특징:

  • 가장 자연스러운 AI 음성
  • 감정 표현 가능
  • 다양한 언어 지원 (한국어 포함)
  • 나만의 목소리 복제 가능 (윤리적 사용 필수!)

내레이션 스크립트 작성 팁

  • ✅ 문장은 짧고 명확하게
  • ✅ 쉼표(,)와 마침표(.)로 자연스러운 호흡
  • ✅ 강조하고 싶은 단어는 대문자로
  • ✅ 숫자는 한글로 (1 → 하나)

🎞️ 5. 최종 프로젝트: 감동 영상 만들기

프로젝트: "소중한 사람에게 보내는 메시지"

1단계: 기획

  • 누구에게 보낼 영상인가? (할머니, 부모님, 친구, 미래의 나)
  • 어떤 메시지를 전할 것인가?
  • 어떤 감정을 전달하고 싶은가?

2단계: 자료 수집

  • 사진 2-3장 선택
  • 스크립트 작성 (200-300자)
  • 배경음악 선택 (저작권 없는 음악)

3단계: AI로 제작

  • D-ID로 립싱크 영상 제작
  • 또는 Runway로 이미지 애니메이션
  • ElevenLabs로 내레이션 추가

4단계: 편집

  • CapCut, iMovie, 또는 DaVinci Resolve 사용
  • 영상 클립 연결
  • 배경음악 추가 (볼륨 조절)
  • 자막 추가 (선택사항)
  • 시작/끝 타이틀 추가

5단계: 완성 및 공유

  • 최종 영상 렌더링 (1080p 권장)
  • 소중한 사람에게 전달
  • 반응과 피드백 기록

🎨 6. 영상 편집 기초 팁

CapCut 편집 가이드

  • 클립 연결: 장면 전환 효과 추가 (Fade, Dissolve)
  • 음악 추가: 볼륨은 대사의 20-30% 수준으로
  • 자막: 중요한 부분만 강조
  • 속도 조절: 감동 장면은 슬로우 모션
  • 색보정: 따뜻한 톤으로 감성 UP

AI로 편집 도움받기

ChatGPT 활용:

1분 30초 감동 영상의 편집 구성을 만들어줘. 재료: - 할머니 사진 3장 - 내레이션 30초 - 배경음악 (피아노) 시간별로 어떤 장면을 넣고, 어떤 효과를 쓸지 알려줘.
