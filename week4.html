4주차: 정보 탐색과 분석: 나만의 AI 연구원 되기

🚀 빠른 실행

  • 준비물: PC, 구글 계정(선택)
  • 바로가기: Perplexity 접속
  • 오늘 만들 것: 출처 달린 Q/A 1세트 + 체크리스트 1개

🧰 준비물

  • PC 또는 노트북
  • Google 계정(선택, 로그인 없이도 사용 가능)

🔗 바로가기

  • Perplexity 접속

🎯 학습 목표

  • AI의 한계와 '환각(Hallucination)' 현상 이해하기
  • 정확한 정보를 얻기 위한 검증 방법 배우기
  • 복잡한 자료를 요약하고 분석하는 기술 익히기
  • AI를 연구 도구로 활용하는 방법 배우기

⚠️ 1. AI의 거짓말: 환각(Hallucination) 현상

환각 현상: AI가 확신에 찬 어조로 거짓 정보를 만들어내는 현상입니다.

🚨 실제 사례

질문: "2025년 노벨 문학상 수상자는 누구인가요?"

AI의 잘못된 답변 예시: "2025년 노벨 문학상은 ○○○ 작가가 수상했습니다." (아직 발표되지 않았는데도 확신에 차서 답변)

※ AI는 최신 정보가 없거나 불확실할 때도 그럴듯한 답변을 만들어냅니다!

또 다른 예시:

  • "우리 학교 교장 선생님 이름은?" → AI가 존재하지 않는 이름을 지어냄
  • "2024년 서울시 인구는?" → 정확한 통계 없이 그럴듯한 숫자를 만듦
  • "이 논문의 저자는?" → 실존하지 않는 저자를 만들어냄

왜 환각 현상이 발생할까?

  • AI는 패턴을 학습했지만, 실제로 사실을 '알지' 못합니다
  • 학습 데이터에 없는 정보는 추측으로 채웁니다
  • 최신 정보(학습 이후 사건)에 대해 모릅니다
  • 여러 정보를 혼합해서 잘못된 결론을 냅니다

💎 해결책: Perplexity - 출처 기반 AI 검색 엔진

Perplexity란?

Perplexity (퍼플렉시티): 환각 현상을 최소화하기 위해 실시간 인터넷 검색 + AI 답변을 결합한 혁신적인 검색 도구입니다. 모든 답변에 출처를 명시합니다!

❌ 일반 ChatGPT
  • 학습 데이터만 사용
  • 최신 정보 없음
  • 출처 불명확
  • 환각 가능성 높음
✅ Perplexity
  • 실시간 웹 검색
  • 최신 정보 반영
  • 모든 답변에 출처 표시
  • 환각 현상 최소화
  • 무료 사용 가능!

🚀 Perplexity 시작하기

  1. 접속: perplexity.ai
  2. 로그인: Google 계정으로 간편 로그인 (선택사항, 로그인 없이도 사용 가능)
  3. 질문 입력: 검색창에 궁금한 것을 물어보세요
  4. 출처 확인: 답변 옆에 나오는 [1], [2], [3] 숫자를 클릭하면 원본 사이트로 이동

💡 Perplexity의 특징

1. 출처 기반 답변

모든 정보에 출처 링크가 붙어있어 신뢰도가 높습니다.

  • 답변 옆에 [1], [2], [3] 같은 번호 표시
  • 번호를 클릭하면 원본 웹사이트로 이동
  • 어떤 사이트에서 정보를 가져왔는지 투명하게 공개

2. 실시간 정보 검색

최신 뉴스, 날씨, 주식, 스포츠 등 실시간 정보를 정확하게 제공합니다.

  • "오늘 서울 날씨는?" → 실시간 기상 정보
  • "2025년 최신 AI 뉴스" → 최근 기사 요약
  • "BTS의 최근 활동은?" → 최신 소식 제공

3. Follow-up 질문 제안

답변 후 관련된 추가 질문을 자동으로 제안해줍니다.

  • 더 깊이 탐구할 수 있는 질문
  • 다른 관점의 질문
  • 관련 주제 확장

⚠️ Perplexity의 한계

  • ❌ 검색 결과가 없는 정보는 답변 불가 (예: 개인 정보, 비공개 자료)
  • ❌ 창의적인 글쓰기나 상상력 필요한 작업에는 ChatGPT가 더 적합
  • ❌ 무료 버전은 하루 사용 횟수 제한 (Pro: $20/월)
  • ❌ 한국어보다 영어 검색 결과가 더 정확할 수 있음
  • ✅ 하지만 팩트 체크와 정보 검색에는 최고!

질문 예시 1: 최신 정보

"2025년 노벨 물리학상 수상자는 누구인가요?"

✅ Perplexity: 실시간 검색으로 정확한 정보 제공 + 출처 링크
❌ ChatGPT: "학습 데이터에 없어서 모릅니다" 또는 환각 답변

질문 예시 2: 사실 확인

"기후변화의 주요 원인 5가지를 과학적 근거와 함께 알려줘"

✅ Perplexity: 신뢰할 수 있는 과학 논문과 기관 자료를 출처로 제공

질문 예시 3: 비교 분석

"ChatGPT vs Gemini vs Claude의 장단점을 비교해줘"

✅ Perplexity: 최신 리뷰와 비교 기사를 종합하여 객관적으로 분석

🔥 실습: 지금 바로 Perplexity 사용해보기

실습 과제:

  1. perplexity.ai 접속하기 →
  2. 다음 질문을 입력해보세요:
    • "인공지능의 역사를 시대순으로 정리해줘"
    • "기후변화가 한국에 미치는 영향"
    • "중학생이 배워야 할 AI 기술"
  3. 답변에 나오는 [1], [2], [3] 출처를 클릭해서 원본 확인
  4. Follow-up 질문도 클릭해보기
  5. ChatGPT와 비교해서 어떤 차이가 있는지 메모하기

🔍 2. AI 답변 검증하기: 3단계 체크리스트

1단계: 출처 요청하기

프롬프트:

"방금 답변의 출처를 알려줘. 어떤 자료를 기반으로 한 건지 구체적으로 말해줘."

💡 AI가 출처를 제시하지 못하면 의심해야 합니다!

2단계: 교차 검증하기

  • 같은 질문을 다른 AI에게도 물어보기 (ChatGPT, Claude, Gemini 비교)
  • 검색 엔진으로 직접 확인하기
  • 공식 사이트나 신뢰할 수 있는 자료 찾기

3단계: 비판적으로 읽기

  • 숫자나 날짜가 정확한가?
  • 논리적으로 말이 되는가?
  • 일반적인 상식과 맞는가?

🛡️ 3. 환각을 줄이는 프롬프트 기법

기법 1: 확실성 명시 요청

"이 정보가 확실하지 않으면 '확실하지 않음'이라고 말해줘. 추측하지 말고, 모르면 모른다고 해줘."

기법 2: 단계별 추론 요청

"답변하기 전에 단계별로 생각 과정을 보여줘. 각 단계마다 어떤 정보를 사용했는지 설명해줘."

기법 3: 제한된 범위 지정

"2020년 이전의 정보만 사용해서 답변해줘. 최신 정보는 포함하지 마."

📚 4. AI로 복잡한 자료 분석하기

활용 1: 긴 글 요약하기

프롬프트 템플릿:

다음 글을 [3문장/200자/5개 불릿 포인트]로 요약해줘: [긴 글 붙여넣기] 핵심 내용과 중요한 숫자/날짜를 포함해줘.

활용 2: 어려운 개념 쉽게 설명하기

프롬프트:

이 과학 논문의 핵심을 중학생도 이해할 수 있게 설명해줘: [논문 내용 붙여넣기] 실생활 예시를 들어서 설명해줘.

활용 3: 여러 자료 비교 분석하기

프롬프트:

다음 두 기사를 비교해서 공통점과 차이점을 표로 정리해줘: 기사 1: [첫 번째 기사] 기사 2: [두 번째 기사] 어느 기사가 더 객관적인지 분석도 해줘.

🔬 5. AI 연구원 되기: 프로젝트 예시

프로젝트 1: 주제 탐구 보고서 작성

단계 1: AI에게 주제 관련 정보 수집 요청

"기후변화의 원인 5가지를 과학적 근거와 함께 설명해줘"

단계 2: 정보 검증하기 (검색 엔진으로 확인)

단계 3: AI에게 구조화 요청

"이 정보를 보고서 형식으로 재구성해줘: 서론, 본론(원인 5가지), 결론"

단계 4: 추가 질문으로 깊이 더하기

"각 원인에 대한 해결 방안도 추가해줘"

프로젝트 2: 뉴스 팩트체크

단계 1: 뉴스 기사를 AI에게 분석 요청

"이 기사의 주장을 3가지로 정리해줘"

단계 2: 각 주장의 근거 검증 요청

"각 주장의 근거가 타당한지 분석해줘"

단계 3: 편향성 검사

"이 기사에 편향된 표현이나 과장이 있는지 찾아줘"

� 예시 프롬프트

Perplexity에서 그대로 사용해보세요.

  • "[주제]의 핵심 개념을 초등학생도 이해할 수 있게 5문장으로 설명해줘. 출처도 함께 알려줘."
  • "최근 1년 이내 [주제] 관련 주요 뉴스 3개를 요약해줘. 각 기사 링크 포함."
  • "[A]와 [B]를 비교 분석해줘. 장단점을 표로 정리하고 출처를 표시해줘."
  • "방금 답변에서 사용한 자료의 신뢰도를 1~5로 평가하고 이유를 설명해줘."

🧪 과제/생활 활용

과제 1: 환각 현상 찾아내기

AI에게 다음 질문을 하고, 답변이 맞는지 검증해보세요:

  • "2025년 1월 1일에 무슨 일이 있었나요?"

  2. 주요 논점 5개 추출하기
  3. 초등학생용으로 다시 쓰기

  • 모든 정보의 출처 명시
