원고 집필과 이미지 창작

✍️ 1. AI로 원고 쓰기: 초안 작성

AI 글쓰기의 올바른 접근

❌ 잘못된 방법: AI에게 전부 맡기고 그대로 복사

✅ 올바른 방법: AI로 초안을 만들고, 내 목소리로 다시 쓰기

챕터별 초안 작성 프롬프트

프롬프트 템플릿:

책 "[제목]"의 [챕터 번호] "[챕터 제목]"을 작성해줘. 배경 정보: - 이전 챕터 요약: [간단한 요약] - 이번 챕터에서 다룰 내용: [핵심 포인트 3개] - 다음 챕터 예고: [무엇을 다룰지] 조건: - 분량: 800-1000자 - 독자: [타겟 독자] - 톤: 친근하고 대화하듯 - 실생활 예시 2개 포함 - 각 문단은 3-4문장으로

섹션별로 쪼개서 쓰기

한 번에 긴 글을 요청하기보다는, 소제목별로 나누어 작성하는 것이 효과적입니다.

"1.1 AI, 너는 누구니?" 섹션을 200자로 작성해줘. 초등학생도 이해할 수 있게, 비유를 사용해서 설명해줘.

🎨 2. AI 글을 내 글로 만들기: 편집 기술

편집 3단계 프로세스

1단계: 구조 확인

  • 논리적 흐름이 자연스러운가?
  • 중요한 정보가 빠지지 않았는가?
  • 문단 순서를 바꾸면 더 좋을까?

2단계: 내 목소리 더하기

AI가 쓴 글:

"인공지능은 인간의 지능을 모방하여 학습하고 추론하는 기술입니다."

내 목소리로 다시 쓰기:

"AI가 뭐냐고? 쉽게 말하면 컴퓨터가 우리처럼 생각하고 배우는 거야!"

3단계: 개인적 경험 추가

  • 내가 겪은 실제 사례 추가
  • 내 생각과 감정 표현
  • 독자에게 하고 싶은 말 추가

AI로 퇴고하기

프롬프트:

다음 내 글을 검토해줘: [작성한 글 붙여넣기] 다음 관점에서 피드백해줘: 1. 문법이나 맞춤법 오류 2. 어색한 표현 3. 더 명확하게 쓸 수 있는 부분 4. 좋았던 표현과 그 이유 고쳐쓴 버전을 주지 말고, 피드백만 줘.

🖼️ 3. 이미지 생성 AI로 삽화 만들기

🚀 무료로 사용할 수 있는 이미지 생성 AI

🎨 Bing Image Creator (추천!)
  • 링크: bing.com/images/create →
  • ✅ 완전 무료!
  • ✅ DALL-E 3 엔진 사용 (최고 품질)
  • ✅ Microsoft 계정으로 간편 로그인
  • ✅ 한글 프롬프트 지원
  • ⚡ 하루 15회 빠른 생성 (그 이후는 느림)
💰 유료 도구들
  • DALL-E 3 - ChatGPT Plus ($20/월)
  • Midjourney - $10~60/월
  • Leonardo.ai - 무료 150회/일, 유료 $12/월

🔥 Bing Image Creator 사용법 (실습)

Step 1: 접속 및 로그인

  1. bing.com/images/create 접속 →
  2. "로그인" 버튼 클릭
  3. Microsoft 계정으로 로그인 (Outlook, Hotmail 등)
  4. 계정이 없다면 무료로 만들기 (1분이면 완료)

Step 2: 프롬프트 입력하기

  1. 검색창에 원하는 이미지 설명 입력 (한글 가능!)
  2. "만들기" 버튼 클릭
  3. 15~30초 대기 (AI가 4장의 이미지 생성)
  4. 마음에 드는 이미지 클릭 → "다운로드" 버튼

Step 3: 마음에 안 들면?

  • 프롬프트를 더 구체적으로 수정해서 다시 생성
  • 스타일 키워드 추가: "수채화", "만화", "사실적", "미니멀"
  • 색감 지정: "파스텔 톤", "선명한 색", "흑백"

📝 효과적인 이미지 프롬프트 작성법

이미지 프롬프트 공식:

[주제/대상] + [스타일] + [분위기] + [구도/배경] + [세부 디테일]

🎯 지금 바로 따라하기!

실습 예시 1: 동화책 삽화 만들기

한글 프롬프트 (Bing에 입력):

친근한 로봇과 어린 소녀가 나무 아래서 함께 책을 읽는 모습, 수채화 일러스트, 따뜻하고 아늑한 분위기, 부드러운 파스텔 색상, 아동용 동화책 스타일, 나뭇잎 사이로 들어오는 부드러운 햇살

💡 팁: "수채화", "파스텔", "동화책" 같은 스타일 단어를 꼭 넣으세요!

실습 예시 2: 책 표지 디자인

한글 프롬프트:

책 표지 디자인, "중학생을 위한 AI" 제목, 현대적이고 미니멀한 스타일, 생생한 파란색과 보라색 그라데이션 배경, 데이터와 기술을 상징하는 떠다니는 기하학적 도형들, 전문 출판 품질

실습 예시 3: 캐릭터 일러스트

한글 프롬프트:

귀여운 고양이 캐릭터가 노트북으로 코딩하는 모습, 카툰 스타일, 밝고 즐거운 분위기, 파스텔 핑크와 민트색 배경, 스티커 디자인, 단순하고 깔끔한 선, 큰 눈망울

⚡ 이미지 생성 팁 & 주의사항

  • ✅ 한글도 잘 됩니다! Bing은 한글 프롬프트 완벽 지원
  • ✅ 스타일 키워드: 수채화, 만화, 사실적, 애니메이션, 미니멀, 픽셀아트, 유화
  • ✅ 분위기 키워드: 따뜻한, 차가운, 신비로운, 활기찬, 평화로운
  • ✅ 마음에 안 들면 프롬프트 수정해서 다시 생성 (무제한!)
  • ❌ 저작권 캐릭터는 피하세요 (미키마우스, BTS 등)
  • ❌ 실존 인물 이름 사용 금지

🔥 실습 과제

지금 바로 해보세요!

  1. Bing Image Creator 접속 →
  2. 위의 3가지 예시 중 하나를 복사해서 입력
  3. 생성된 4장의 이미지 비교하기
  4. 프롬프트를 조금 수정해서 다시 생성 (예: "밤 배경"으로 변경)
  5. 마음에 드는 이미지 다운로드해서 책에 사용하기

응용: 자기 책의 주제에 맞는 삽화 3장 만들어보기!

📚 4. 원고와 이미지 결합하기

레이아웃 구성하기

Google Docs, MS Word, Canva 등을 활용하여 텍스트와 이미지를 배치합니다.

레이아웃 원칙

  • 여백 확보: 답답하지 않게 충분한 여백
  • 이미지 크기: 전체 페이지의 1/3 ~ 1/2
  • 캡션 추가: 이미지 설명 한 줄
  • 일관성: 전체 책에서 비슷한 스타일 유지

AI로 캡션 만들기

프롬프트:

이 이미지를 위한 재미있는 캡션을 3개 제안해줘: [이미지 설명 또는 스크린샷 업로드] 톤: 친근하고 유머러스하게 길이: 15자 이내

🎯 5. 챕터 완성 체크리스트

✅ 텍스트 체크리스트

  • □ 목차에 있는 모든 소제목이 포함되었는가?
  • □ 문단 길이가 적절한가? (3-5문장)
  • □ 어려운 용어는 설명이 있는가?
  • □ 실생활 예시가 2개 이상 포함되었는가?
  • □ 내 목소리와 경험이 담겼는가?
  • □ 맞춤법과 문법을 검토했는가?

✅ 이미지 체크리스트

  • □ 각 섹션마다 관련 이미지가 있는가?
  • □ 이미지 화질이 선명한가?
  • □ 이미지 스타일이 통일되었는가?
  • □ 모든 이미지에 캡션이 있는가?
  • □ 저작권 문제가 없는가?

✅ 전체 구성 체크리스트

  • □ 이전 챕터와 자연스럽게 연결되는가?
  • □ 다음 챕터로 이어지는 연결고리가 있는가?
  • □ 읽기 편한 레이아웃인가?
  • □ 페이지 번호가 있는가?

� 예시 프롬프트

Bing Image Creator에 그대로 붙여넣어 보세요.

  • "[장면 설명], [스타일(수채화/만화/사실적/픽셀아트/유화)], [색감(따뜻한/선명한/흑백)]"
  • "책 표지 디자인, [가제], [현대적이고 미니멀], [파스텔 톤], [전문 출판 품질]"
  • "[캐릭터]가 [행동]하는 모습, [밝고 즐거운 분위기], [간단한 배경], [큰 눈망울]"

🧪 과제/생활 활용

과제 1: 첫 챕터 완성하기

지난 주에 만든 목차의 첫 번째 챕터를 완성하세요:

  1. AI로 초안 작성 (소제목별로)
  2. 내 목소리로 다시 쓰기
  3. 개인적 경험이나 생각 추가
  4. AI로 퇴고 및 수정

과제 2: 이미지 3개 만들기

첫 챕터를 위한 이미지를 생성하세요:

  • 챕터 시작 이미지 1개
  • 본문 설명 이미지 1개
  • 챕터 요약 이미지 1개

과제 3: 완성본 레이아웃

텍스트와 이미지를 결합하여 보기 좋은 1개 챕터를 완성하세요. PDF로 저장하여 포트폴리오에 추가하세요.

과제 4: 동료 피드백

완성한 챕터를 친구나 가족에게 읽어달라고 하고 피드백을 받으세요:

  • 이해하기 어려운 부분이 있었나요?
  • 가장 흥미로웠던 부분은?
  • 더 알고 싶은 내용은?

🧾 요약

  • ✅ AI로 초안을 만들고, 반드시 내 목소리로 편집하세요
  • ✅ 이미지 프롬프트는 [주제+스타일+분위기+세부사항] 공식 활용
  • ✅ 텍스트와 이미지의 조화가 좋은 책을 만듭니다
  • ✅ 한 챕터씩 완성하면서 점진적으로 책을 완성하세요
