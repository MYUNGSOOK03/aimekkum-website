7주차: AI 데이터 분석가 되기
설문과 차트 만들기

🚀 빠른 실행

  • 준비물: PC, 구글 계정
  • 바로가기: Google Forms · Google Sheets · ChatGPT
  • 오늘 만들 것: 10문항 설문 1개, 응답 차트 1개, CSV 분석 요약 1개

🧰 준비물

  • Google 계정(Forms/Sheets 사용)
  • CSV 파일 뷰어(엑셀/스프레드시트)

🔗 바로가기

  • Google Forms · Google Sheets · ChatGPT

🎯 학습 목표

  • 데이터 분석의 기본 프로세스 이해하기
  • AI로 효과적인 설문 문항 만들기
  • 데이터를 시각화하는 방법 배우기
  • ChatGPT의 데이터 분석 기능 활용하기

📊 1. 데이터 분석이란?

데이터 분석: 숫자와 정보를 수집하고, 패턴을 찾아 의미 있는 결론을 도출하는 과정입니다.

데이터 분석 5단계

  1. 질문 정의: 무엇을 알고 싶은가?
  2. 데이터 수집: 설문, 관찰, 기존 자료 활용
  3. 데이터 정리: 표나 스프레드시트로 정리
  4. 분석: 패턴과 인사이트 찾기
  5. 시각화: 차트나 그래프로 표현

❓ 2. AI로 설문 문항 만들기

좋은 설문의 조건

  • ✅ 명확하고 이해하기 쉬운 질문
  • ✅ 중립적인 표현 (유도 질문 X)
  • ✅ 적절한 질문 수 (너무 길면 피로감)
  • ✅ 다양한 질문 유형 혼합

AI로 설문 문항 생성하기

프롬프트 템플릿:

"[주제]"에 관한 설문을 만들고 싶어. 목적: [설문의 목적] 대상: [응답자 - 예: 중학생 100명] 다음 조건으로 설문 문항 10개를 만들어줘: 1. 객관식 5개 (선택지 4-5개) 2. 척도형 3개 (1-5점 또는 1-10점) 3. 단답형 2개 각 문항마다 왜 이 질문이 필요한지 설명도 해줘.

실전 예시: 학생 학습 습관 조사

객관식 문항:

1. 하루 평균 공부 시간은?

  • ① 1시간 미만
  • ② 1-2시간
  • ③ 2-3시간
  • ④ 3-4시간
  • ⑤ 4시간 이상

척도형 문항:

2. AI 도구(ChatGPT 등)를 공부에 활용하는 정도는? (1점: 전혀 안 함 ~ 5점: 매우 자주)

1 --- 2 --- 3 --- 4 --- 5

단답형 문항:

3. 공부할 때 가장 큰 어려움은 무엇인가요? (한 문장으로)

🚀 무료 설문 도구 실습

1️⃣ Google Forms (추천!)

링크: docs.google.com/forms →

  • ✅ 완전 무료, 무제한
  • ✅ 자동 집계 및 차트 생성
  • ✅ Google Sheets로 바로 연동

사용법:

  1. 위 링크 접속 → "빈 설문지" 클릭
  2. 제목 입력 (예: "우리 반 AI 활용 조사")
  3. "+" 버튼으로 질문 추가
  4. 질문 유형 선택: 객관식, 체크박스, 단답형, 척도
  5. 오른쪽 상단 "보내기" → 링크 복사해서 친구들에게 공유
  6. "응답" 탭에서 실시간 결과 확인

2️⃣ Naver Form

링크: form.naver.com →

  • ✅ 네이버 계정만 있으면 OK
  • ✅ 한국어 UI, 사용 편리

🔥 실습 과제: 설문 만들기

지금 바로 만들어보세요!

  1. Google Forms 접속 →
  2. 위의 "학생 학습 습관 조사" 3개 문항을 그대로 만들기
  3. 친구 5명에게 링크 보내서 응답 받기
  4. "응답" 탭에서 자동 생성된 차트 확인하기

🗂 예시 프롬프트

Forms/ChatGPT에서 그대로 사용해보세요.

  • "[주제]에 대한 10문항 설문을 만들어줘. 객관식 5, 척도 3, 단답 2로 구성하고, 각 문항의 목적도 설명해줘."
  • "아래 CSV 설문 응답을 분석해서 주요 통계, Top 3 인사이트, 추천 행동 3가지를 불릿으로 정리해줘."
  • "이 데이터로 막대그래프 1개와 원형그래프 1개가 적합한 항목을 추천해줘."

📈 3. AI로 데이터 분석하기

💎 ChatGPT 데이터 분석 기능 (무료!)

ChatGPT는 파일을 업로드하면 자동으로 분석해줍니다! (GPT-4o 모델 사용 - 무료 계정도 가능!)

✅ 무료 계정으로도 가능한 방법

  1. chatgpt.com 접속 →
  2. Google Forms 응답 → "Google Sheets에서 보기" 클릭
  3. Sheets에서 파일 → 다운로드 → CSV (.csv) 선택
  4. ChatGPT에서 📎 클립 아이콘 클릭 → CSV 파일 업로드
  5. 아래 프롬프트 입력!

📝 복사해서 사용할 프롬프트

기본 분석 프롬프트:

첨부한 설문 데이터를 분석해줘. 다음을 알려줘: 1. 전체 응답자 수 2. 각 질문의 응답 분포 (비율과 개수) 3. 가장 많이 선택된 답변 top 3 4. 흥미로운 패턴이나 인사이트 5. 막대 그래프와 원 그래프로 시각화해줘

심화 분석 프롬프트:

학년별로 응답이 어떻게 다른지 비교해줘. 그룹 막대 그래프로 보여줘.
"공부 시간"과 "AI 도구 활용"에 상관관계가 있는지 산점도로 분석해줘.
이 데이터를 보고 3가지 결론과 제안사항을 요약해줘.

🔥 실습: 데이터 분석 체험

따라하기:

  1. Google Forms에서 만든 설문 응답을 CSV로 다운로드
  2. ChatGPT 접속 →
  3. 📎 클립 아이콘으로 CSV 파일 업로드
  4. 위의 "기본 분석 프롬프트" 복사해서 입력
  5. 생성된 차트 저장하기 (마우스 우클릭 → 이미지 저장)
  6. 심화 분석 프롬프트도 시도해보기!

📊 4. 데이터 시각화: 차트 만들기

🎨 Google Sheets로 차트 만들기 (가장 쉬움!)

Step-by-Step:

  1. Google Forms 응답 → "Sheets에서 보기" 클릭
  2. 차트로 만들 데이터 영역 드래그로 선택
  3. 상단 메뉴: 삽입 → 차트
  4. 오른쪽 "차트 편집기"에서 차트 유형 선택
  5. 색상, 레이블 등 커스터마이징
  6. 차트를 클릭 → 오른쪽 위 ⋮ → 이미지로 다운로드

📌 차트 유형 선택 가이드

  • 막대 그래프: 항목별 비교 (예: 선호도 순위)
  • 원 그래프 (파이 차트): 전체 대비 비율 (예: 학년별 분포)
  • 선 그래프: 시간에 따른 변화 (예: 월별 성적)
  • 산점도: 두 변수의 관계 (예: 공부 시간 vs 성적)

💡 온라인 차트 도구 추천

🎯 초보자 추천
  • Google Sheets: sheets.google.com - 자동 차트, 가장 쉬움
  • Canva: canva.com - 예쁜 디자인 템플릿
🔥 고급 사용자
  • Flourish: flourish.studio - 인터랙티브 차트
  • DataWrapper: datawrapper.de - 언론사 수준

🏆 최종 프로젝트 아이디어

우리 반/학교 조사 프로젝트:

  • 🎮 "중학생 게임 시간 vs 수면 시간 분석"
  • 📱 "우리 반 SNS 사용 실태 조사"
  • 🤖 "AI 도구 인식 및 활용도 조사"
  • 📚 "독서 습관과 성적의 상관관계"
  • 🍕 "급식 메뉴 선호도 조사"

💡 직접 설문 만들고 → 응답 수집하고 → 분석하고 → 차트 만들어서 발표!

📝 5. 분석 보고서 작성하기

보고서 구조

1. 제목 및 요약

  • 설문 주제
  • 조사 기간 및 대상
  • 핵심 발견 1-2줄 요약

2. 조사 방법

  • 응답자 수
  • 설문 방식
  • 문항 수

3. 주요 결과

  • 문항별 결과 (차트 포함)
  • 흥미로운 패턴
  • 예상 밖의 결과

4. 결론 및 제언

  • 데이터가 말해주는 것
  • 실행 가능한 제안

AI로 보고서 초안 만들기

프롬프트:

다음 분석 결과를 바탕으로 보고서를 작성해줘: [분석 결과 붙여넣기] 형식: - 제목, 조사 개요, 주요 결과, 결론 포함 - 각 섹션마다 불릿 포인트 활용 - 중학생도 이해할 수 있는 표현 - A4 2페이지 분량

🧪 과제/생활 활용

프로젝트: 우리 반 데이터 분석

1단계: 주제 선택 (다음 중 하나)

  • 우리 반 학생들의 취미와 관심사
  • AI 도구 사용 현황
  • 독서 습관 조사
  • 진로 희망 조사

2단계: AI로 설문 문항 10개 만들기

  • 객관식, 척도형, 단답형 혼합
  • Google Forms로 설문 생성

3단계: 데이터 수집

  • 최소 20명 이상 응답 받기
  • CSV 파일로 내보내기

4단계: AI로 분석 및 차트 생성

  • ChatGPT에 데이터 업로드
  • 분석 요청 및 차트 3개 이상 생성

5단계: 보고서 작성

  • A4 2-3페이지 분량
  • 차트 포함
  • 인사이트와 제언 포함

🧾 요약

  • ✅ 데이터 분석은 질문 → 수집 → 정리 → 분석 → 시각화 순서
  • ✅ AI로 효과적인 설문 문항을 빠르게 만들 수 있습니다
  • ✅ ChatGPT는 데이터 파일을 직접 분석하고 차트를 만들 수 있습니다
  • ✅ 시각화는 복잡한 데이터를 한눈에 이해하게 만듭니다
