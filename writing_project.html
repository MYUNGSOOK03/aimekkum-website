AI 글쓰기 에이전트

목적과 문체를 설정하면 맞춤형 프롬프트를 자동 생성해주는 글쓰기 도우미입니다.
ChatGPT, 뤼튼 등 AI 도구에 바로 붙여넣어 고품질 글쓰기를 시작하세요.

🚀 빠른 실행

  • 준비물: PC 또는 스마트폰
  • 바로가기: 맞춤 프롬프트 생성기로 즉시 시작
  • 오늘 볼 것: 프롬프트 생성 → ChatGPT/뤼튼에 붙여넣기
✍️ 에이전트 실행 💬 ChatGPT 열기 ✨ 뤼튼 열기

📋 프로젝트 개요

🎯 프롬프트 생성기

• 글 주제, 대상 독자, 문체 선택
• 목적, 화자, 격식도 설정
• 변환/편집 옵션 체크
• 자동으로 완성된 프롬프트 생성
👉 직접 사용해보기

📚 실제 활용

• 자서전/회고록 초안 작성
• 블로그 포스팅 (일기→에세이)
• 업무 문서 (보고서 문체 조정)
✓ 5-6주차 교육 연계

🎯 주요 기능

  • 맞춤형 선택 옵션: 글쓰기 맥락, 문체, 대상 독자, 목적, 화자/시점, 격식도 선택
  • 변환/편집 옵션: 내용 확장, 요약/축약, 재작성, 번역, 교정 등 후처리 옵션
  • 자동 프롬프트 생성: 선택한 옵션을 바탕으로 구조화된 프롬프트 자동 작성
  • 원클릭 복사: 생성된 프롬프트를 클립보드에 복사하고 AI 도구로 바로 이동

🔄 워크플로

1️⃣ 글 주제 입력 → 작성하고 싶은 주제 설정
2️⃣ 옵션 선택 → 맥락(에세이/블로그/칼럼 등), 문체, 대상 독자, 목적 선택
3️⃣ 세부 설정 → 화자/시점, 격식도, 분량, 구성 지정
4️⃣ 변환 옵션 → 필요시 확장/축약/번역 등 추가 작업 체크
5️⃣ 프롬프트 생성 → 자동으로 완성된 프롬프트 확인
6️⃣ 복사 & 실행 → ChatGPT/뤼튼에 붙여넣어 즉시 실행

💡 활용 시나리오

📖 시나리오 1: 자서전 초안 작성

설정: 주제 "어린 시절의 추억", 맥락 "자서전", 문체 "진지하고 성찰적인", 대상 "중장년(40-60세)", 분량 "800자"
결과: 구조화된 3단락 초안 + 핵심 메시지 + 개선 포인트

✏️ 시나리오 2: 일기를 블로그로 변환

설정: 주제 "오늘의 산책", 맥락 "블로그", 문체 "친근하고 따뜻한", 변환옵션 "재작성/문체변환" 체크
결과: 개인 일기가 독자 친화적 블로그 포스팅으로 변환

📄 시나리오 3: 업무 보고서 작성

설정: 주제 "분기 실적 보고", 맥락 "보고서", 문체 "간결하고 명확한", 대상 "전문가", 격식 "격식있는"
결과: 전문적이고 간결한 업무 보고서 양식

🗂 예시 프롬프트

  • "너는 친근한 에세이 코치야. 주제는 '가을 산책'. 500자 초안을 3단락으로 작성하고 마지막에 한 줄 교훈을 넣어줘."
  • "방금 초안을 '신문 칼럼' 문체로 300자로 축약해줘."
  • "자기검토 체크리스트 6개를 만들어서 내 글을 점검해줘 (문장 호흡, 중복어, 퇴고 포인트 포함)."
  • "친구에게 보낼 편지 형식으로, 따뜻하고 진솔한 문체로 '감사의 마음' 전달하는 글 800자 작성해줘."

🧰 사용 기술

프롬프트 엔지니어링 체계적 워크플로 설계 문체 변환 기법 HTML/CSS/JavaScript 클라이언트 사이드 UI

🎨 설계 원칙

  • 사용자 중심: 중장년층(40-60세)이 쉽게 선택할 수 있는 직관적 UI
  • 체계적 구조: 목적 → 대상 → 문체 → 세부옵션 순서로 단계적 입력
  • 즉시 실행: 생성된 프롬프트를 바로 복사하고 AI 도구로 이동 가능
  • 교육 연계: 5-6주차 커리큘럼과 실습 예제 통합

🎯 결과/성과

  • ✅ 15개 이상의 선택 옵션으로 맞춤형 프롬프트 생성
  • ✅ ChatGPT, 뤼튼 등 다양한 AI 도구와 호환
  • ✅ 프롬프트 템플릿 기반으로 누구나 재현 가능
  • ✅ 5-6주차 교육 커리큘럼과 실습 연계 완료
  • ✅ 자서전, 블로그, 업무문서 등 다양한 장르 적용 검증

💡 활용 팁

  • 단계적 접근: 처음에는 기본 옵션만 사용하고, 익숙해지면 변환 옵션 추가
  • 반복 개선: 첫 결과물을 받은 후 "더 친근하게" 등 추가 지시로 다듬기
  • 키워드 활용: 핵심 키워드를 2-3개 입력하면 주제가 더 명확해짐
  • 문체 실험: 같은 주제로 여러 문체를 시도해보며 가장 적합한 스타일 찾기
